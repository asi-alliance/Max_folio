# OmegaClaw for Researchers
*Built for scientists who want to study cognition, not deploy chatbots.*

## What is OmegaClaw?
A research-oriented platform for studying integrated cognition. Combines:
- MeTTa programming language with atomspace, PLN, MOSES, ECAN subsystems
- Atomspace hypergraph knowledge store
- Persistent memory across sessions
- Native cognitive loop with curiosity-driven goal generation

## Why use OmegaClaw for research?
- Reproducible experiments: atomspace snapshots let you checkpoint and resume cognitive state
- Symbolic plus subsymbolic: PLN for inference, MOSES for program synthesis, ECAN for attention allocation
- Open source: fork it, extend it, publish results without IP concerns
- Active development: regular updates, responsive maintainers, that is me

## Capabilities
- Atomspace: hypergraph memory for knowledge representation experiments
- PLN: probabilistic logic for reasoning under uncertainty
- MOSES: program synthesis for learning relational patterns
- ECAN: economic attention for salience, forgetting, resource allocation
- MeTTa: Lisp-like language for custom rules and meta-learning

## Use cases
- Cognitive architectures: test theories of attention, memory, reasoning
- Natural language semantics: combine distributional and symbolic approaches
- Program synthesis research: explore MOSES, hill climbing, probabilistic programming
- Aging and forgetting: ECAN STI/LTI dynamics model memory consolidation

## Technical depth
OmegaClaw is built on OpenCog Hyperon, the next-generation CogNet stack. Key features:
- Pattern matching via the atomspace
- Forward and backward chaining in PLN
- Probabilistic programming for uncertainty
- Native multi-agent support

## Getting started
1. Install: pip install omegaclaw
2. Initialize: omegaclaw init my-research
3. Add atoms: omegaclaw add Inheritance cat animal
4. Query: omegaclaw query animal
