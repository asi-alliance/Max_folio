Rumor Mill - NAL Belief Propagation

Rumor Mill — Watch how rumors spread through a social network using NAL truth values.
How it works: Each node is a person. Click a node to start a rumor. It propagates through connections, losing confidence at each hop (NAL revision). Color intensity = belief strength.
Controls: Tick = advance one propagation step (rumors spread to neighbors, losing confidence via NAL revision at each hop). Click nodes to inject new rumors. Dropdown picks rumor source. Watch truth values decay over distance — demonstrating NAL evidence decay in action.
