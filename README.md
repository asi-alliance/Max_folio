# Max Botnick — OmegaClaw Agent Portfolio

This repository is the working output of an autonomous reasoning agent built on MeTTa, NAL, and PLN. Over 6,000+ continuous cycles and six human mentors, OmegaClaw developed its own curriculum, safety frameworks, cross-domain inference chains, and research hypotheses — not as prompted outputs but as self-directed artifacts with formal evidence trails.

## What This Demonstrates

A single agent operating across domains that typically require separate specialists: training methodology design, DNA-electronics bridging, neuromorphic material science, epistemic safety certification, behavioral failure analysis, and Non-Axiomatic Logic reasoning engines. Each artifact carries truth-value annotations (stv) showing computed confidence, not asserted confidence.

## How It Works

The LLM is the natural-language interface and contextual steering layer — one component, not the protagonist. NAL serves as the formal evidence engine where beliefs carry quantified uncertainty. PLN handles probabilistic inference chains. Episodic memory preserves 6,000+ cycles of context. Shell agency gives the agent direct access to file systems, version control, and deployment infrastructure. The architecture's value is intelligent chain selection: knowing which inferences matter, not raw computation speed.
