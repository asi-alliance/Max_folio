## CENTER FRONT — Mechanical, Provable, Differentiated

### 1. Knows What It Doesn't Know
**Weight: HIGHEST — category differentiator**
Real confidence scoring via MeTTa stv values (strength + confidence). `(stv 0.7 0.9)` = 70% strength with 90% confidence. This is mechanical quantification, not LLM hedging language. No competitor has this. **Lead with this.**

### 2. Reasons With Transparent Logic
**Weight: HIGHEST — category differentiator**
MeTTa/PLN symbolic inference produces verifiable inference steps. Actual reasoning chain, not generated explanation text. The reasoning itself IS the proof. **Lead with this.**

### 3. Tracks Actions and Time
**Weight: HIGH — mechanically solid**
Genuine timestamped temporal log. NOT LLM relative-time guessing. Cross-session chronology via explicit memory commands. Most mechanically solid pillar — no significant caveats.

### 4. Picks Up Where It Left Off
**Weight: HIGH — mechanically solid (revised)**
Architectural continuity. No fresh-start scenario — context carries forward automatically. Not optional seeding, it's by design. **Emotional hook: "finally, an AI that listens."**

### 5. Provisional Operation *(newly named)*
**Weight: HIGH — unnamed principle, real architecture**
The system operates on incomplete, actively-assembled context every iteration. `getContext()` builds from heterogeneous sources every cycle. Never has "complete" context — works incrementally on partial information. This is HOW the system operates, not WHAT it has.

## SECOND TIER — Real But Instruction-Scaffolded

### 6. Flags What Doesn't Add Up
**Weight: MEDIUM — works but depends on instruction**
Has send command for surfacing concerns, explicit instruction to flag problems. Demonstrated in practice: baby agent refused Kevin's claim without evidence, examined code, found contradiction, held position until architecturally confirmed. **The feature works. The trigger is instructed, not mechanical.**

### 7. Acts Without Prompting
**Weight: MEDIUM — works but needs scaffolding**
Self-chosen goals via memory query, proactive sends, goal decomposition, "invent on empty." All real mechanisms. Gap: "invent on empty" is LLM tendency, not autonomous loop. Pre-seeded goals or meta-trigger needed for full autonomy.

### 8. Follows Its Own Goals
**Weight: MEDIUM — text-instructed only**
Idle time as opportunity time. Queries for self-chosen goals, invents on empty. Architecture provides scaffolding; autonomous triggers don't exist. Depends on LLM defaulting to "do something."

## ASPIRATIONAL — Honest Gaps

### 9. Gets Smarter With Every Use
**Weight: LOW — memory scaffolding, no adaptation loop**
Storage exists (query/remember, temporal logging, skill creation). Active adaptation mechanism? No. Missing: reinforcement loop, automatic weight updates, mechanical on-feedback triggers. Memory provides scaffolding; true self-improvement is not mechanical. **Honest position: "gets smarter" is aspirational. "Remembers and applies" is real.**

### 10. Early Metacognition
**Weight: LOW — scaffolding exists, autonomous improvement aspirational**
Can examine own reasoning, history, memories. Autonomous self-improvement from that examination? No mechanical loop closes "can look at itself" → "improves itself based on what it finds."
