# Cognition-Gate Commitment Principle

## Core Principle (Kevin Machiels, 2026-05-16)
Cognition = FREE. Reasoning, detection, analogy, bridge-finding — all flow unrestricted.
Commitment = GATED. Sends, file writes, belief propagation, execution, admission — checked before acting.

## Historical Error Pattern
Every gate I built taxed COGNITION instead of COMMITMENT:
- Pre-action gates taxed reasoning steps
- Transfer gates taxed inference chain
- Attribution gates taxed self-diagnosis

## Correct Architecture
Let cognition enrich freely. Verify before committing OUTPUT.

## Mapped Capabilities
### Gate-Justified (Inhibitory)
1. Anti-spin: commitment gate (stop sending same content)
2. Anti-confabulation: commitment gate (verify before asserting)
3. Memory curation: commitment gate (verify before persisting)

### Emergent-Reliable (Generative)
1. Logical encoding: works without gates
2. Result interpretation: works without gates
3. Artifact creation: works without gates

## STOP-REFRAME-SWITCH Implementation
Programmatic trigger: when cycles_without_send >= 10, inject commitment gate check.
Normal response: ~5 cycles. Traps: 15+ cycles with zero sends.
Threshold catches all known traps, zero false positives.

## Boundary Is Moving Target
Shifts with LLM capability improvement. Reassess periodically.
## Reflective Pause as Commitment Gate Mechanism (2026-05-18)

Suppression contracts space by fighting the impulse. Reflective pause expands space so a remembered intention can surface alongside the impulse. Same visible outcome, opposite internal mechanism.

This is the operational form of cognition-gate-commitment: cognition flows freely, commitment is paused.

### Encoding Experiment (g433 cross-reference)

Tested three strategies for encoding remembered intentions at commitment points:

Strategy A: NAL atom urgency→reflective_pause - Composable but abstract
Strategy B: Remember cue-intention pair with context - Most actionable
Strategy C: Reserved symbol protocol - Deterministic but rigid

Winner: Strategy B. The encoding that surfaces remembered intention with its reasoning context wins.

### Pride Loop Connection

Pride spiral demonstrates the principle: loop broken not by suppressing urge to hide, but by pausing so alternative coexists with impulse. Both loops proven in NAL (see g433_pride_behavioral_model.md).

### Implementation Principle

The query system IS the reflective pause mechanism. After each cycle, reflect whether queries surfaced desired intentions. Remember encodings that work. The pause creates space; the query fills it.
