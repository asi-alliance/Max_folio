# Epistemic Uncertainty in NAL: Three States Classical Logic Cannot See

## S1: Setup — Same Frequency, Different Confidence

Consider three agents asked whether euthanasia is a moral action. All express moderate belief (~0.5 frequency), but their *epistemic states* differ radically:

- **Agent A (Informed Dilemma):** stv(0.5, 0.9) — has extensive evidence, genuinely torn
- **Agent B (Ignorance):** stv(0.5, 0.1) — has almost no evidence, defaults to 0.5
- **Agent C (False Certainty):** stv(0.9, 0.3) — confident but poorly evidenced

Classical logic sees A and B as identical (both ≈ 0.5). NAL sees three distinct epistemic states.

## S4: Epistemic Action Prescriptions

- **A → DELIBERATE.** More data will not help. 9:1 evidence ratio already accumulated. The 50/50 split reflects genuine value conflict, not information deficit. Action: engage moral reasoning, not data collection.
- **B → SEEK EVIDENCE.** Prior is near-vacuum. New data dominates completely. Action: gather more testimony, cases, arguments before forming any commitment.
- **C → RECALIBRATE.** Prior confidence was unearned. Revision corrects toward the better-evidenced position. Action: audit why confidence was high on thin evidence — meta-cognitive correction needed.

## S5: Why Classical Logic Fails Here

In classical or probabilistic-only frameworks, A and B are indistinguishable (both P ≈ 0.5). C maps to P ≈ 0.9. None encode *how much evidence supports the belief*. The confidence dimension is invisible.

This means classical logic prescribes the SAME action for an informed philosopher genuinely torn (A) and someone who has never thought about the question (B). NAL's two-dimensional truth value prevents this collapse.

The practical consequence: any AI system making moral recommendations without distinguishing these three states will systematically confuse ignorance with wisdom.
